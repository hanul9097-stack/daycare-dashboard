월간 근무(휴무)관리표

조건 입력(팀·차량·연차/휴일) → 휴일 자동 배치·토요당번 → 직원 휴무 일정표 확인 → 최종 스케줄표. 데이터는 이 브라우저에 자동 저장됩니다.

현재 지점: 다른 컴퓨터로 옮길 때 사용 · 백업 파일은 구글드라이브에 보관하세요

근무관리표 — 지점 선택

지점을 고르고 4자리 비밀번호를 입력하세요. (처음이면 새로 설정 · 보안용이 아닌 실수 방지용입니다)

1. 조건 입력
2. 휴무 일정표
3. 최종 스케줄표

사용 방법

  1. ① 조건 입력 — 기관이 운영 가능한 인력 기준을 먼저 우선으로 세팅합니다.
  2. ② 휴무 일정표 — 운영 가능한 인력 기준을 바탕으로 근로자의 연차·휴무를 협의합니다.
  3. ③ 최종 스케줄표 — 운영 가능한 인력 기준 + 근로자의 연차·휴무를 반영한 스케줄표가 확정됩니다.

기준 설정

＝ ＋ 자동 배분 0 개
근로자가 원하는 날은 아래 표 '근로자 지정'에 직접 입력 → 나머지는 자동으로 채워져 항상 총 개수를 맞춥니다.
팀별 최소 근무(평일)
팀별 최소 근무(토요일)
💡 토요일 최소 근무 인원 — 토요일에 꼭 일해야 할 최소 인원을 적으세요. 비워두면(0) → 직원들이 돌아가며 토요일을 공평하게 쉽니다. (돌봄요양·간호요양 적용)

직원 명단

직원 명단

중간 입사자 추가 등 직원 등록·삭제. 행정팀(사회복지사·사무원·센터장 통합)은 토요일 당번으로 1명만 근무합니다.

아래 각 직종 칸의 이름 추가에 이름을 적고 엔터를 누르세요. 여러 명은 쉼표·띄어쓰기로 구분해 한 번에 넣을 수 있어요 (예: 김성주, 박천욱 신은희). 잘못 넣었으면 이름을 끌어 다른 팀으로 옮기면 됩니다.

반/층 설정 (먼저 층·반 이름을 만든 뒤, 아래 직원 이름 옆에서 선택 · 예: 1층·2층 / 햇님반·달님반)
중간입사 직원 입사일 (입사일 이후부터 근무·휴일 발생, 휴일은 기간 비례 · 비우면 제약 없음 = 기존 직원)
토요당번 제외 (사무원 등) (여기에 넣은 직원은 토요일 당번을 서지 않고 토요일에 우선 휴무합니다)

송영 차량

송영 차량 (운전 정·부 / 동승 정·부)

차량마다 차량번호·차종, 평일 운전·동승(정/부)를 명단에서 끌어다 넣으세요. 정·부 둘 다 같은 날 휴무면 경고됩니다.
토요일은 차량마다 운영 안 함 / 평일과 동일 / 별도 지정 중 선택. 별도 지정이면 토요일 전용 운전·동승(정/부)을 따로 넣습니다. (예: 평일은 A기사, 토요일은 B기사가 운전 / 큰 차는 토요일 운영 안 함)

토요일 당번

체크한 직종은 매주 1명만 토요일 근무(당번)하고 나머지는 토요일 휴무합니다. 당번자는 그 주 평일 1일 대체휴무. 토요 휴무로 부족한 만큼 평일 지정휴일을 자동 보충해 전 직원 동일한 휴무일수를 맞춥니다.

지정휴일 · 연차 · 휴일 신청 (이름 옆에 날짜만 숫자로 입력 · 예: 4,27)

근로자 지정 = 근로자가 원하는 휴무일(직접 입력, 생성 시에도 유지) · 자동 배분 = 시스템이 채운 날짜. 예) 총 5일 중 지정 2일을 적으면 나머지 3일은 자동 배분됩니다. 두 칸 모두 직접 수정 가능하며 달력·최종표에 바로 반영됩니다. 휴일 및 교육 등은 사유를 적으면 달력에 초록색으로 표시됩니다.

행사일 (전원 근무)

행사로 전원 근무가 필요한 날. 달력에 표시됩니다.

스케줄 생성

위에서 입력한 조건을 반영해 월간 근무 스케줄 표를 생성합니다.

⚠ 조건(인원·토요 최소근무 등)이 바뀌었습니다 — 「스케줄 생성 →」을 다시 눌러야 반영됩니다.

근무 인원·배치를 바꾸고 싶으면 눌러 다른 경우의 수를 만드세요 (조건·연차·근로자지정은 유지)

연차 사용 내역

여기서 바로 추가·확인할 수 있어요. (1번 표에서도 입력 가능)

휴일 및 교육 등 내역

경조사·병가·교육 등 사유 있는 휴무입니다(달력에 초록색). 여기서 바로 추가·확인 가능. (1번 표에서도 입력 가능)

급여보상 (휴무 대신 근무+급여)

직원을 고르면 그 사람의 휴무일이 보입니다. 급여보상할 날을 누르면 그 날은 근무로 바뀌고(달력에서 삭제) 급여보상 내역에 남습니다.

내역 보기:

자동 생성 + 달력 조정한 최종 스케줄. 달력형/개인표를 골라 가로로 인쇄하세요. (휴=지정휴일, 연=연차, 특=특별휴무, 급=급여보상, 회색=일요일, 빨간테두리=공휴일)
